# NAVBC Learning

[![MIT License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Contributions Welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg)](CONTRIBUTING.md)

Open source learning platform for **Microsoft Dynamics 365 Business Central** developers.

**Live:** [learn.navbc.com](https://learn.navbc.com) · **Contribute:** [/contribute](https://learn.navbc.com/contribute) · **GitHub:** [taher-el-mehdi/navbc-learning](https://github.com/taher-el-mehdi/navbc-learning)

## Quick start

```bash
git clone https://github.com/taher-el-mehdi/navbc-learning.git
cd navbc-learn
npm install
npm run dev
```

## Contribute

- Fix or add lessons in `content/` (MDX — no React needed)
- [Report issues](https://github.com/taher-el-mehdi/navbc-learning/issues/new/choose)
- [Join discussions](https://github.com/taher-el-mehdi/navbc-learning/discussions)
- Read **[CONTRIBUTING.md](./CONTRIBUTING.md)**

Every page includes a contribute bar at the bottom with GitHub links.

## Project structure

```
content/           # Courses & lessons (main contribution area)
src/               # Next.js platform code
docs/              # Contributor docs
.github/           # Issue & PR templates
```

## Tech stack

Next.js 15 · TypeScript · Tailwind CSS · MDX · Shiki
